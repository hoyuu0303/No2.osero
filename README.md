#説明書
- No2.pyをひらいてコードをコピーする。
- Visual studio codeを開く。
- Visual studio code内でファイル＞新しいテキストファイルへいってクリックする。
- コピーしたコードをVisual studio code内に貼り付けする。
-  画面右上に三角マークのボタンが表示されているため、クリックする。
-  オセロができる。
-  白が先攻
-  黒が後攻
-どちらかがおけなくなるか、盤面を埋めたら勝ち
