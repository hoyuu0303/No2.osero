# No2.osero
#開発環境
- 動作プラットフォーム
- **WIN11**
- 開発プラットフォーム
- **Visual studio code**
- 開発言語
- **Python**
- 利用ライブラリ
- **tkinter**

#説明書
- **No2.py**をひらいてコードを**コピー**する。
- **Visual studio code**を開く。
- Visual studio code内でファイル＞新しいテキストファイルを作成する。
- コピーしたコードをVisual studio code内に**貼り付け**る。
- 画面右上に**三角マーク**のボタンが表示されているため、クリックする。
-  オセロができる。
-  白が先攻
-  黒が後攻
-  どちらかが置けなくなるか盤面が埋まったら、数が多い方が勝ち
